🎨 Nova UI 元件庫

統一的按鈕、表單、表格、Badge、Toggle 元件，避免重複 inline style

按鈕 (UI.btn)
主要按鈕
UI.btn.dark('儲存', 'save()')
搜尋按鈕
UI.btn.search('搜尋', 'search()')
輪廓按鈕
UI.btn.outline('取消', 'cancel()')
新增按鈕（綠色）
UI.btn.add('新增', 'add()')
次要按鈕（淡藍）
UI.btn.secondary('清除', 'clear()')
危險按鈕
UI.btn.danger('刪除', 'del()')
圖示按鈕
UI.btn.icon('edit', 'edit()', '編輯')
UI.btn.icon('delete', 'del()', '刪除')
UI.btn.icon('upload', 'upload()', '上傳')
Badge 標籤
UI.badge('使用中', 'on')
UI.badge('停用中', 'off')
UI.badge('維護中', 'maint')
UI.badge('即將上線', 'soon')
UI.statusBadge('使用中') // 自動判斷
Toggle 開關
UI.toggle('on', 'toggle()')
UI.toggle('off', 'toggle()')
Segmented Control 分段控制
UI.segmented([
{label: '金幣', active: true, onclick: 'switchCurr("gold")'},
{label: '星幣', active: false, onclick: 'switchCurr("star")'}
])
提示框
Tips Box
UI.tips('廳級設定優先覆蓋個別遊戲')
表單元件 (UI.form)
下拉選單
UI.form.select('hall', '娛樂廳', ['VA', 'YGR', 'JDB'], {required: true})
文字輸入
UI.form.input('name', '遊戲名稱', {placeholder: '請輸入...', hint: '最多 20 字'})
日期時間
UI.form.datetime('start', '開始時間', {required: true})
兩欄排列
UI.form.row(
UI.form.input('min', '最低投注', {type: 'number'}),
UI.form.input('max', '最高投注', {type: 'number'})
)
表格 (UI.table)
UI.table.create(
[{label: '序號', width: '60px'}, {label: '名稱'}, {label: '狀態', width: '80px'}],
[['1', '埃及三秘寶', UI.statusBadge('使用中')], ['2', '財神倍倍發', UI.statusBadge('停用中')]],
{empty: '無資料'}
)
Toast 通知
UI.toast('操作成功', 'success')
UI.toast('操作失敗', 'error')
UI.toast('請注意', 'warning')
SVG 圖示 (UI.icon)
UI.icon.clock
UI.icon.edit
UI.icon.search
UI.icon.sort
UI.icon.save
UI.icon.image
UI.icon.drag
UI.icon.bulb
UI.icon.monitor
分頁 (UI.pagination)
完整分頁（左側每頁筆數 + 右側頁碼）
UI.pagination({
containerId: 'paginationDemo',
currentPage: 2,
totalPages: 5,
onPageChange: 'goToPage',
pageSize: 20,
pageSizes: [20, 50, 100]
})
純頁碼（無每頁筆數）
UI.pagination({
containerId: 'paginationDemo2',
currentPage: 1,
totalPages: 3,
onPageChange: 'goToPage'
})
訂單編號 & 幣別
訂單編號（灰底圓角）
UI.orderId('019e1b33-242e-74bc-b283-7ad99eb398e7')
幣別符號
UI.coinGold() // ⊙ 金幣
UI.coinStar() // ⊙ 星幣
UI.coinGold('30,000') // 自訂文字
